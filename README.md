Linear Algebra with Python

This repository contains Python implementations of Linear Algebra concepts with interactive examples and visualization.

Features
- Matrix operations (add, multiply, inverse, determinant)
- Vector operations (dot, cross, projection)
- Eigenvalues & Eigenvectors (power method, diagonalization)
- Linear Transformations (2D/3D visualization)
- Jupyter notebooks for step-by-step learning

Tech Stack
- Python 3.11
- NumPy
- Matplotlib / Plotly
- Jupyter Notebook

Getting Started
1. Clone the repo
2. Install dependencies: 
3. Run notebooks to explore concepts
